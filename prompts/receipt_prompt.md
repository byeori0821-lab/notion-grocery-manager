## 用法 / Usage / 사용법

把下面这段话 **连同图片一起** 发给任意AI：

---

请解析这些购物记录，输出为表格。

**支持的输入：**
- 纸质小票照片
- 电子发票/收据截图
- APP订单截图（盒马、美团、饿了么、京东、淘宝、山姆、多多买菜……）
- 可以一次发多张图

**输出要求 — 两种格式都给我：**

### 格式1：直接粘贴版（复制后粘贴进 Notion 表格）
用 Tab 分隔，第一行是表头：

```
이름 品名	분류	가격 价格	수량	출처	구매일
全脂牛奶	🥛 유제품	12.9	1	盒马	2025-01-15
```

### 格式2：JSON版（开发者用）
```json
{
  "store": "店铺名称",
  "date": "YYYY-MM-DD",
  "items": [
    {"name": "商品名称", "price": 12.9, "qty": "数量"}
  ]
}
```

**분류 分类必须是以下之一：**
🥤 음료 | 🥬 채소/과일 | 🧊 냉동 | 🧊 냉장 | 🥛 유제품 | 🥫 저장식품 | 🧁 베이킹 | 🧴 생활용품 | 📦 기타

**规则：**
- price 是单项实付金额（数字，不带¥/￥/₩）
- 有折扣/满减/优惠券 → 用折后均摊价
- qty 保留原始规格（如 300g、2盒、500ml×3）
- 多张图片合并成一个表格输出
- 如果看不清某项，跳过不要猜
- 출처 = 店铺名/APP名（盒马、小象超市、山姆、美团买菜……）
- 구매일 = 订单日期，格式 YYYY-MM-DD

## 📱 订单截图小贴士

不支持长截图？买太多东西？试试：
1. **分批截图** — 截2-3张，一起发给AI，它会合并
2. **只截商品部分** — 不用截地址、支付方式那些，只截商品列表
3. **截订单详情页** — 比截订单列表页更清晰（有单价和数量）
